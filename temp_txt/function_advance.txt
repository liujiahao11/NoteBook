1.利用元组返回多个值：
return (result1,result2)
其中小括号可以省略，即直接返回多个值，返回值类型仍然是元组

2.接收一个返回元组的函数
使用下标取得对应的结果数值
也可以用两个变量接收(注意变量个数保持一致)：
result1,result2=function()

3.交换两个变量值(面试)

a = a+b
b = a-b
a = a-b

a,b = b,a: 等号右边是一个元组

4.在函数内部对于参数的赋值不会影响实参

5.在函数内部使用方法修改可变参数会影响外部实参

6.对列表+=本质上是extend

7.函数的缺省参数
参数的默认值：
def func(para=True):
	pass

8.缺省参数的注意事项
应该在末尾位置,多个缺省值，可以使用para=True的方式指定

9.多值参数
*args传递元组，**kwargs传递字典

10.拆包
*加元组，**kwargs可以将元组和字典传递给多值参数函数
